# Changelog — 2026-03-29

## สิ่งที่ทำวันนี้

### 1. Claude AI Messenger Bot (แทน n8n)
- สร้าง `claude-messenger.service.js` — ใช้ Claude Sonnet API ตรงจาก backend
- ตัด n8n ออกจาก flow ทั้งหมด — webhook ส่งตรงไป Claude
- AI อ่านข้อมูลจากระบบได้: ลูกค้า, ออเดอร์, สินค้า, ราคา, ประวัติแชท
- สกัดข้อมูลลูกค้าอัตโนมัติ (ชื่อ, จังหวัด, เบอร์โทร)

### 2. Multi-Page Support (รองรับหลายเพจ Facebook)
- สร้างตาราง `facebook_pages` — เก็บ token, AI settings แยกแต่ละเพจ
- PageSelector component — เลือกเพจจาก sidebar
- ทุกหน้าแสดงข้อมูลเฉพาะเพจที่เลือก (Messenger, Orders, Customers, Products)

### 3. AI Mode (3 โหมด)
- **ปิด** — AI ไม่ตอบใครเลย
- **ทดสอบ** — AI ตอบเฉพาะ Tester (พนักงานที่ผูก Facebook)
- **ใช้งานจริง** — AI ตอบลูกค้าทุกคน
- ค่าเริ่มต้น = ทดสอบ เพื่อความปลอดภัย

### 4. Facebook Apps
- สร้างแอพ **HATZ Production** (ID: 1345963747555331) สำหรับ Messenger webhook
- สร้างแอพ **HATZ Login** (ID: 1482154660268433) สำหรับ Facebook Login (Consumer type)
- เชื่อมเพจ Hat Fix & Clean + webhook 32 ช่อง

### 5. Admin Panel UX Redesign
- **ตั้งค่า AI** — redesign ด้วย 3-mode selector, AI persona, collapsible system prompt, preview chat
- **Profile Page** — Facebook Login popup (JS SDK), แก้ไขข้อมูล, เปลี่ยนรหัสผ่าน
- **Messenger Page** — แชทแยกตามเพจ, admin ตอบเอง / สั่ง AI ตอบ
- **Page Selector** — dropdown เลือกเพจ พร้อม avatar + AI status badge

### 6. Backend APIs ใหม่
- `GET/PUT /api/users/me/profile` — โปรไฟล์ user
- `PUT /api/users/me/facebook` — เชื่อมต่อ Facebook
- `PUT /api/users/me/password` — เปลี่ยนรหัสผ่าน
- `POST /api/facebook-pages/:id/ai-mode` — เปลี่ยนโหมด AI
- `GET /api/products/summary` — สินค้ารวม + แยกเพจ
- `GET/POST /api/conversations` — จัดการแชท Messenger
